Today I learnt

I guess going from Windows to Linux can be painful for a lot of people. However, I am so glad that, more than ten years ago, my previous job introduced me to the open-sourced, free operating system, that gives all the power to the user. I still use Windows for software like Photoshop or Comic Studio. But I'm really waiting for the day when even these will have a Linux version, so that I don't have to use Windows anymore.

That said, can you imagine the opposite? Today I had to move definitely from Ubuntu to a Mac (at least for work). It's not as moving from Linux to Windows, which would have been worse (at least Mac is also a Unix-based system!).

I had never used a Mac before, but Jim wants me to use one, and everybody in the team uses one. Skim is probably a weird exception, since he uses Linux in a virtual box running on Mac, because he also prefers Linux, but, as he explained to me, Mac is superior in terms of hardware. Changing from my Mountain to a Mac will also be better for pairing, since the Spanish layout of my laptop's keyboard seemed to be driving everybody crazy.

The first thing I missed was the terminal. Mac comes with a terminal, but in Ubuntu, it is a SPACESHIP that makes you feel so powerful. Mac's term didn't even let me use the tab key to auto-complete commands! Also, the user experience trolls the user a lot. Like, when in full-screen mode, the top menu dissapears, and so does the bottom toolbar. So suddenly, you are at Limbo, and you have to waste precious time hovering the region until said bars appear. The hovering takes time, the appearing too. The solution? Exit full-screen, and resize your window to occupy the whole screen without covering the toolbar. Because there's no such thing as setting something like an "always visible" state for them in Mac. YOU CAN'T BE SERIOUS, STEVE JOBS. Y U DO DIS TO US.

Finder also gave me some frustrations, until I discovered that things just had a hide and show state, so I activated "show" on everything. There's also nothing like Baobab. Jim showed me how typing command + spacebar would pop a modal search box pop-up where I can type the name of an application to run. But I am more of a tree-view kind of person. I'm all about hierarchies of folders and relationships between files. I like to know what I have installed and where. I want to move through the tree in microseconds. WAT. DON'T LOOK AT ME LIKE THAT. I'm a control freak. Ok?

Jim showed me brew as well, which is the equivalent of apt-get in Ubuntu... and doesn't come pre-installed by default! OMG HOW DO PIPOL INSTALL THINGS. He also told me to install cask, so now I install everything through brew cask install whatevah.

Noticing my hardcore-linux-user frustration, Jim told me Mac is a consumer-oriented product, something your grandma would use, so that's why nothing comes pre-installed. But... doesn't that make it like Windows? Ok, maybe not. Windows really treats their users as if they were retarded. But Ubuntu was also created so that it could be used by non-tech-savvy people, and yet it comes with every basic thing. Ah, systems, systems.

As I expected, changing to another system was not straight-forward with bits of frustration, but I decided to start being positive and called it a learning curve. Some of the weapons I needed during my battle:

  • Shift + Delete = Command + Delete
  • Alt + F4 = Command + W
  • F5 = Command + R (WAAA....?)
  • Show desktop: fn + F11
  • Cut and paste, or Ctrl + X followed by Ctrl + V. Surprisingly, there is no such thing as "cut" in a Mac, you have to copy, and then paste in a special way: Command + C followed by Command + Option + V
  • There's also no right-click. You have to click with two fingers in horizontal position several times until you find the "right" position in the pad that would produce the right-click (I told, you, trolling the user).
  • Safari! Aggghhh. I installed Firefox and Chrome and felt like a decent person again.

Together with the browsers, I installed everything else I needed like the JVM, IntelliJ, iTerm, which is a decent terminal, Sublime, which is a decent text editor with syntax highlighting (Ubuntu comes with one pre-installed, BTW, mwahhwahwa), a password manager, and all the things.

My expectations are set on Monday to be able to start using this Mac normally and be productive. I guess now I can call myself a hipster. So, that would make me a hippie wannabe hipster with a Mac.

Done with my rant, let's talk about some nice things.

Nice things
