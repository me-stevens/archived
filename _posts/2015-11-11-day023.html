Today I learnt

In today's Zagaku, Daniel talked about the tree created by conditionals, and how that is used by languages like SQL to build the query expressions, for example, the conditions coming after the WHERE clause.

Also, he explained how in functional languages, you are closer to the way this tree is generated in the computer, since the operator comes before the operands, just as it happens in the generated tree.

Finally, I was having some problems when returning things from an array in my BoardTest class, and that lead me to learn something new from Felipe. I had this test:

@Test
public void getsTheEmptyCellIndexes() {
	board.setCell(1, "hi");

	ArrayList expected = new ArrayList();
	for (int index = 2; index <= 9; index++) {
		expected.add(index);
	}

	assertEquals(expected, board.getEmptyCellIndexes());
}

I wanted to test the functionality of the getEmptyCellIndexes() method in my Board class, so I placed a mark on the first cell of the board and then asserted that the array returned from that method was equal to the expected that I was building manually. Since my method returns the indexes of the cells of the board that are empty, my expected array was storing all indexes from 2 to 9 (only cell of index 1 was busy with a mark).

For some weird reason, the expected array was returning a 6, and my method was returning the right indexes, so I had an assertion fail. If I changed the condition to end in any other number that was not 9, I got a proper array list that looked like this: [2, 3, 4, 5, 6, 7, 8] (for a loop condition ending in 8). But if I put back the 9, I got a 6 again. Where was that 6 coming from? Neither Felipe or Christoph nor I were able to grasp the origin of that 6. However, Felipe showed me a way to initialize an ArrayList that doesn't require a for loop and can be done in one line:

@Test
public void getsTheEmptyCellIndexes() {
	board.setCell(1, "hi");

	List expected = Arrays.asList(2,3,4,5,6,7,8,9);

	assertEquals(expected, board.getEmptyCellIndexes());
}

And suddenly, my test was green. It's cool to learn new things! The 6 though, will forever remain a mistery.
